# JavaScript Functions

## 1. Add Task

The `addTask()` function creates a new task and appends it to the task list.

```javascript
function addTask(){

    let input=document.getElementById("taskInput");

    if(input.value===""){
        alert("Please enter a task");
        return;
    }

    let li=document.createElement("li");

    li.innerHTML=input.value;

    document.getElementById("taskList").appendChild(li);

    input.value="";
}
```

---

## 2. Delete Task

The Delete button removes the selected task.

```javascript
deleteButton.onclick=function(){

    li.remove();

};
```

---

## 3. Complete Task

Clicking on a task marks it as completed.

```javascript
li.onclick=function(){

    li.classList.toggle("completed");

};
```

# DOM Methods Used

| Method | Purpose |
|---------|----------|
| `getElementById()` | Selects an HTML element |
| `createElement()` | Creates a new HTML element |
| `appendChild()` | Adds an element to the webpage |
| `classList.toggle()` | Adds or removes a CSS class |
| `remove()` | Deletes an element |
| `value` | Reads the value entered by the user |

---

# Working Process

1. User enters a task.
2. Clicks the **Add Task** button.
3. JavaScript creates a new `<li>` element.
4. The task is displayed on the webpage.
5. Clicking the task marks it as completed.
6. Clicking the Delete button removes the task.
