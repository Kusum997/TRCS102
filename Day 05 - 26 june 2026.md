# JavaScript Functions

## What is a Function?

A **function** is a reusable block of code that performs a specific task. Instead of writing the same code multiple times, we can create a function and call it whenever needed.

### Syntax

```javascript
function functionName(){
    // Code to execute
}
```

---

## Function to Find Sum

```javascript
function sum(a, b){
    return a + b;
}

console.log(sum(10, 20));
```

**Output**

```text
30
```

---

## Function to Find Square

```javascript
function square(num){
    return num * num;
}

console.log(square(5));
```

**Output**

```text
25
```

---

## Function to Find Cube

```javascript
function cube(num){
    return num * num * num;
}

console.log(cube(3));
```

**Output**

```text
27
```

---

## Arrow Function

An **Arrow Function** is a shorter way to write functions in JavaScript. It is introduced in ES6.

### Syntax

```javascript
const functionName = () => {
    // Code
};
```

### Example

```javascript
const greet = () => {
    console.log("Welcome to JavaScript");
};

greet();
```

**Output**

```text
Welcome to JavaScript
```

### Arrow Function with Parameters

```javascript
const add = (a, b) => {
    return a + b;
};

console.log(add(15, 25));
```

**Output**

```text
40
```

# JavaScript Arrays

## What is an Array?

An **array** is a collection of multiple values stored in a single variable. Each value in an array is called an **element**, and every element has an **index** starting from **0**.

### Syntax

```javascript
let fruits = ["Apple", "Banana", "Mango"];
```

---

## Accessing Array Elements

```javascript
let fruits = ["Apple", "Banana", "Mango"];

console.log(fruits[0]);
console.log(fruits[1]);
```

**Output**

```text
Apple
Banana
```

---

# Adding Elements to an Array

## 1. `push()` Method

The `push()` method adds one or more elements **at the end** of an array.

```javascript
let fruits = ["Apple", "Banana"];

fruits.push("Mango");

console.log(fruits);
```

**Output**

```text
["Apple", "Banana", "Mango"]
```

---

## 2. `unshift()` Method

The `unshift()` method adds one or more elements **at the beginning** of an array.

```javascript
let fruits = ["Banana", "Mango"];

fruits.unshift("Apple");

console.log(fruits);
```

**Output**

```text
["Apple", "Banana", "Mango"]
```

---

# Removing Elements from an Array

## 1. `pop()` Method

The `pop()` method removes the **last element** from an array.

```javascript
let fruits = ["Apple", "Banana", "Mango"];

fruits.pop();

console.log(fruits);
```

**Output**

```text
["Apple", "Banana"]
```

---

## 2. `shift()` Method

The `shift()` method removes the **first element** from an array.

```javascript
let fruits = ["Apple", "Banana", "Mango"];

fruits.shift();

console.log(fruits);
```

**Output**

```text
["Banana", "Mango"]
```
