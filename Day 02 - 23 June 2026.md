# Introduction to HTML Forms

## What is an HTML Form?
An **HTML Form** is a section of a webpage designed to collect user inputs and send that data to a web server for processing. Forms are essential for interactive websites, enabling functionalities such as user registration, account logins, search bars, checkout systems, feedback collection, and file uploads.

---

## Basic Form Structure
All forms are created using the `<form>` element, which acts as a container for various input controls (like text fields, checkboxes, and buttons). 

Here is the fundamental syntax for a standard HTML form:

```html
<!DOCTYPE html>
<html>
<head>
    <title>HTML Forms</title>
</head>
<body>

    <h2>Registration Form</h2>

    <form action="/submit" method="POST">
        <label for="username">Name:</label><br>
        <input type="text" id="username" name="username"><br><br>

        <label for="useremail">Email:</label><br>
        <input type="email" id="useremail" name="useremail"><br><br>

        <button type="submit">Submit</button>
    </form>
```
---

## Common Form elements

* `<form>`:Creates a form
* `<input>`:Takes user input
* `<label>`:Adds a label for an input field
* `<textarea>`:Multi-line text input
* `<select>`:Creates a drop-down list
* `<option>`:Defines an option in a drop-down
* `<button>`:Creates a clickable button
* `<fieldset>`:Groups related form elements
* `<legend>`:Gives a title to a fieldset
---

## Common Input Types

* text – Single-line text
* password – Hidden password
* email – Email address
* number – Numeric input
* tel – Phone number
* date – Date picker
* Select one option
* checkbox – Select multiple options
* file – Upload a file
* submit – Submit the form
* reset – Reset the form

# HTML Checkbox
A **checkbox** in HTML allows users to select one or more options from a list. Unlike radio buttons, multiple checkboxes can be selected at the same time.

`<input type="checkbox">`

```<!DOCTYPE html>
<html>
<head>
    <title>Checkbox Example</title>
</head>
<body>

    <h2>Select Your Hobbies</h2>

    <form>

        <input type="checkbox" id="reading" name="hobby" value="Reading">
        <label for="reading">Reading</label><br>

        <input type="checkbox" id="sports" name="hobby" value="Sports">
        <label for="sports">Sports</label><br>

        <input type="checkbox" id="music" name="hobby" value="Music">
        <label for="music">Music</label><br><br>

        <input type="submit" value="Submit">
        <input type="reset" value="Reset">

    </form>

</body>
</html>
```

# HTML Radio Button (type="radio")
A **radio button** in HTML allows the user to select only one option from a group. If one option is selected, the others are automatically deselected.

**Syntax**
`<input type="radio" name="gender" value="Male">`

```<!DOCTYPE html>
<html>
<head>
    <title>Radio Button Example</title>
</head>
<body>

    <h2>Select Your Gender</h2>

    <form>

        <input type="radio" id="male" name="gender" value="Male">
        <label for="male">Male</label><br>

        <input type="radio" id="female" name="gender" value="Female">
        <label for="female">Female</label><br>

        <input type="radio" id="other" name="gender" value="Other">
        <label for="other">Other</label><br><br>

        <input type="submit" value="Submit">
        <input type="reset" value="Reset">

    </form>

</body>
</html>
```

# CSS(Cascading style sheet):
**CSS (Cascading Style Sheets)** is used to style HTML pages. It controls the color, font, size, spacing, layout, and appearance of web pages.

# Why Use CSS?
* Makes web pages attractive
* Changes colors and fonts
* Controls spacing and layout
* Creates responsive designs
* Separates content (HTML) from design (CSS)

# Types of CSS
1. **Inline CSS**
CSS is written inside the HTML tag using the style attribute.
`<p style="color: blue;">Hello World</p>`

2. **Internal CSS**
CSS is written inside the <style> tag in the <head> section.

```<!DOCTYPE html>
<html>
<head>
    <link rel="stylesheet" href="style.css">
</head>
<body>

    <h1>Hello CSS</h1>

</body>
</html>
```
3. **3. External CSS**
CSS is written in a separate .css file and linked to the HTML file.

## HTML File

```<!DOCTYPE html>
<html>
<head>
    <link rel="stylesheet" href="style.css">
</head>
<body>

    <h1>Hello CSS</h1>

</body>
</html>
```
## style.css

```h1 {
    color: green;
    text-align: center;
}
```
## Basic CSS Syntax

```selector {
    property: value;
}
```
## Example

```h1 {
    color: blue;
    font-size: 30px;
}
```
