# Project 1 – Theme Toggle

## What is Theme Toggle?

A **Theme Toggle** allows users to switch between **Light Mode** and **Dark Mode**. It is commonly used in modern websites to improve user experience.

---

## HTML

```html
<!DOCTYPE html>
<html>
<head>
    <title>Theme Toggle</title>
    <link rel="stylesheet" href="style.css">
</head>
<body id="body">

<h1>Theme Toggle</h1>

<button onclick="changeTheme()">
Toggle Theme
</button>

<script src="script.js"></script>

</body>
</html>
```

---

## CSS

```css
body{
    background-color:white;
    color:black;
    text-align:center;
    font-family:Arial;
    transition:0.5s;
}

.dark{
    background-color:black;
    color:white;
}
```

---

## JavaScript

```javascript
function changeTheme(){

    let body=document.getElementById("body");

    body.classList.toggle("dark");

}
```

---

## Output

- Changes the webpage from **Light Mode** to **Dark Mode**.
- Clicking the button again switches back to **Light Mode**.

# Project 2 – Image Toggle

## What is Image Toggle?

Image Toggle changes one image into another when the user clicks a button. It demonstrates DOM manipulation by changing the `src` attribute of an image.

---

## HTML

```html
<!DOCTYPE html>
<html>
<head>
    <title>Image Toggle</title>
</head>
<body>

<img
id="image"
src="image1.jpg"
height="250"
width="250">

<br><br>

<button onclick="changeImage()">
Change Image
</button>

<script src="script.js"></script>

</body>
</html>
```

---

## JavaScript

```javascript
function changeImage(){

    let image=document.getElementById("image");

    if(image.src.includes("image1.jpg")){
        image.src="image2.jpg";
    }
    else{
        image.src="image1.jpg";
    }

}
```

---

## Output

- Initially displays the first image.
- Clicking the button changes it to the second image.
- Clicking again changes it back.

# Project 3 – Counter Application

## What is a Counter?

A Counter Application is a simple JavaScript project that increases or decreases a numeric value using buttons.

---

## HTML

```html
<!DOCTYPE html>
<html>
<head>
    <title>Counter</title>
</head>
<body>

<h1 id="count">0</h1>

<button onclick="increase()">
Increase
</button>

<button onclick="decrease()">
Decrease
</button>

<button onclick="reset()">
Reset
</button>

<script src="script.js"></script>

</body>
</html>
```

---

## JavaScript

```javascript
let count=0;

function increase(){

    count++;

    document.getElementById("count").innerHTML=count;

}

function decrease(){

    count--;

    document.getElementById("count").innerHTML=count;

}

function reset(){

    count=0;

    document.getElementById("count").innerHTML=count;

}
```

---

## Output

- **Increase** button increments the counter.
- **Decrease** button decrements the counter.
- **Reset** button sets the counter back to zero.

# DOM Methods Used

| Method | Purpose |
|---------|---------|
| `getElementById()` | Selects an HTML element by its ID |
| `innerHTML` | Changes the content of an element |
| `classList.toggle()` | Adds or removes a CSS class |
| `src` | Changes the image source |
