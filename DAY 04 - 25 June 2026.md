# Day 4 – JavaScript Conditional Statements (25 June)

## Introduction

Conditional statements are used to make decisions in JavaScript. They allow a program to execute different blocks of code based on whether a specified condition is **true** or **false**. These statements help create dynamic and interactive applications by controlling the flow of execution.

---

## What are Conditional Statements?

A conditional statement evaluates an expression and executes a block of code depending on the result.

For example:
- If a student's marks are greater than or equal to 40, display **Pass**.
- Otherwise, display **Fail**.

---

## Types of Conditional Statements

### 1. `if` Statement

The `if` statement executes a block of code only when the condition is true.

```javascript
let age = 20;

if(age >= 18){
    console.log("Eligible to Vote");
}
```

**Output**

```text
Eligible to Vote
```

---

### 2. `if...else` Statement

The `else` block executes when the condition is false.

```javascript
let marks = 35;

if(marks >= 40){
    console.log("Pass");
}
else{
    console.log("Fail");
}
```

**Output**

```text
Fail
```

---

### 3. `if...else if...else`

Used when multiple conditions need to be checked.

```javascript
let marks = 82;

if(marks >= 90){
    console.log("Grade A+");
}
else if(marks >= 75){
    console.log("Grade A");
}
else if(marks >= 60){
    console.log("Grade B");
}
else if(marks >= 40){
    console.log("Grade C");
}
else{
    console.log("Fail");
}
```

**Output**

```text
Grade A
```

---

### 4. Nested `if`

A nested `if` statement is an `if` statement placed inside another `if` statement. It is used when multiple conditions must be checked in sequence.

```javascript
let age = 20;
let citizen = true;

if(age >= 18){
    if(citizen){
        console.log("Eligible to Vote");
    }
}
```

**Output**

```text
Eligible to Vote
```

## Comparison Operators Used in Conditions

| Operator | Meaning |
|----------|---------|
| `==` | Equal to |
| `===` | Strict Equal to |
| `!=` | Not Equal to |
| `>` | Greater Than |
| `<` | Less Than |
| `>=` | Greater Than or Equal to |
| `<=` | Less Than or Equal to |

---

## Logical Operators

| Operator | Meaning |
|----------|---------|
| `&&` | Logical AND |
| `\|\|` | Logical OR |
| `!` | Logical NOT |

### Example

```javascript
let age = 19;
let hasID = true;

if(age >= 18 && hasID){
    console.log("Entry Allowed");
}
```

**Output**

```text
Entry Allowed
```
