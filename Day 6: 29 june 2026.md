# Advanced Array Methods

JavaScript provides several built-in methods to perform operations on arrays without writing lengthy code.

---

## 1. `filter()` Method

The `filter()` method creates a **new array** containing only the elements that satisfy a given condition.

### Syntax

```javascript
array.filter(function(element){
    return condition;
});
```

### Example

```javascript
let numbers = [10, 20, 30, 40, 50];

let result = numbers.filter(num => num > 25);

console.log(result);
```

**Output**

```text
[30, 40, 50]
```

---

## 2. `find()` Method

The `find()` method returns the **first element** that satisfies a given condition.

### Syntax

```javascript
array.find(function(element){
    return condition;
});
```

### Example

```javascript
let numbers = [10, 20, 30, 40, 50];

let result = numbers.find(num => num > 25);

console.log(result);
```

**Output**

```text
30
```

---

## 3. `slice()` Method

The `slice()` method returns a **new array** by copying a portion of the original array. It **does not modify** the original array.

### Syntax

```javascript
array.slice(startIndex, endIndex);
```

### Example

```javascript
let fruits = ["Apple", "Banana", "Mango", "Orange"];

let result = fruits.slice(1, 3);

console.log(result);
console.log(fruits);
```

**Output**

```text
["Banana", "Mango"]
["Apple", "Banana", "Mango", "Orange"]
```

---

## 4. `splice()` Method

The `splice()` method is used to **add**, **remove**, or **replace** elements in an array. It **modifies the original array**.

### Syntax

```javascript
array.splice(startIndex, deleteCount, item1, item2);
```

### Example

```javascript
let fruits = ["Apple", "Banana", "Mango"];

fruits.splice(1, 1);

console.log(fruits);
```

**Output**

```text
["Apple", "Mango"]
```

---

# Difference Between `slice()` and `splice()`

| `slice()` | `splice()` |
|-----------|------------|
| Returns a new array. | Modifies the original array. |
| Does not change the original array. | Changes the original array. |
| Used for copying a portion of an array. | Used to add, remove, or replace elements. |
| End index is not included. | Requires delete count and optional new elements. |

---

# Multiplying Array Elements Using `map()`

The `map()` method creates a new array by applying a function to every element.

```javascript
let numbers = [1, 2, 3, 4];

let result = numbers.map(num => num * 2);

console.log(result);
```

**Output**

```text
[2, 4, 6, 8]
```

# JavaScript Objects

## What is an Object?

A **JavaScript Object** is a collection of properties, where each property consists of a **key** and a **value**. Objects are used to represent real-world entities and store related information.

### Syntax

```javascript
let student = {
    name: "Kusum",
    age: 19,
    course: "B.Tech CSE"
};
```

---

## Accessing Object Properties

Properties of an object can be accessed using **dot notation** or **bracket notation**.

### Using Dot Notation

```javascript
let student = {
    name: "Kusum",
    age: 19
};

console.log(student.name);
console.log(student.age);
```

**Output**

```text
Kusum
19
```

### Using Bracket Notation

```javascript
console.log(student["name"]);
```

---

## Adding a New Property

```javascript
let student = {
    name: "Kusum",
    age: 19
};

student.city = "Ludhiana";

console.log(student);
```

---

## Updating a Property

```javascript
student.age = 20;

console.log(student);
```

---

## Deleting a Property

```javascript
delete student.city;

console.log(student);
```

# DOM (Document Object Model)

## What is DOM?

The **Document Object Model (DOM)** is a programming interface for HTML documents. It represents the webpage as a tree-like structure of objects, allowing JavaScript to access and modify HTML elements dynamically.

Using the DOM, JavaScript can:

- Change text content.
- Change CSS styles.
- Add or remove HTML elements.
- Handle user events like button clicks.

---

## Selecting Elements

### `getElementById()`

Selects an element using its unique ID.

```html
<h1 id="heading">Hello World</h1>
```

```javascript
let element = document.getElementById("heading");

console.log(element);
```

---

## Changing Text Content

```html
<h1 id="heading">Hello</h1>
```

```javascript
document.getElementById("heading").innerHTML = "Welcome to JavaScript";
```

---

## Changing CSS Using JavaScript

```javascript
document.getElementById("heading").style.color = "blue";
document.getElementById("heading").style.fontSize = "30px";
```

---

## Button Click Example

### HTML

```html
<h2 id="message">Hello</h2>

<button onclick="changeText()">
Click Here
</button>
```

### JavaScript

```javascript
function changeText(){
    document.getElementById("message").innerHTML = "Welcome!";
}
```
